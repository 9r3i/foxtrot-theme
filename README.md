# foxtrot-theme
A theme for Force Website



# Config
Add this to config file for theme section
```json
{
  "theme": {
    "namespace": "foxtrot",
    "host": "https://9r3i.github.io/foxtrot-theme",
    "config": {
      "data": {
        "limit": 8,
        "foot": 55
      }
    }
  }
}
```
The ```config.data.foot``` is the data ID for focontent. And make sure the type of data is "base64".
